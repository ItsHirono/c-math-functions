# c-math-functions

## Research Question

Write a C programming language function to solve the quadratic equation.

## Findings

1. The discriminant (b² − 4ac) determines whether the equation has two real roots, one real root, or two complex roots.
2. The program uses functions from the math.h library, such as sqrt() for square roots and fabs() for absolute values.
3. Input validation helps prevent users from entering invalid data and improves program reliability.
